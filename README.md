# TinySteam
WIP Clean white/orange skin for steam

# Usage

    1. Extract the skin to *\Steam\skins\ (Example: C:\Program Files (x86)\Steam\skins)

    2. Go to Steam -> Settings -> Interface -> Select "TinySteam" from dropdown menu and restart steam
