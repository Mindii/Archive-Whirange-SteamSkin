# Whirange
WIP white/orange skin for steam

# Usage

    1. Extract the skin to *\Steam\skins\ (Example: C:\Program Files (x86)\Steam\skins)

    2. Go to Steam -> Settings -> Interface -> Select "Whirange-skin" from dropdown menu and restart steam
